About the study

Technology built for clearer answers

The Automated Tomographic Breast Ultrasound (ATBUS) Prospective Clinical Case Collection (PCC) study collects imaging data to improve how ultrasound technology captures, processes, and interprets breast tissue — helping clinicians detect potential problem areas with greater confidence and precision.

📸
Clearer image capture
Enhanced ultrasound acquisition for sharper, more detailed breast imaging.
🖥
Advanced image processing
Improved algorithms that refine raw scan data into high-resolution diagnostic images.
🤖
AI-powered tissue analysis
Machine learning tools trained to identify tissue types and flag areas of clinical interest.

About Aperia Medical

Advancing breast health through AI-powered imaging

Aperia Medical is working to improve ATBUS technology by making image capturing clearer, processing the images better, and using AI-powered tools to identify different types of tissue and spot potential problem areas.

Aperia Medical is advancing breast imaging technology within an FDA PMA-approved (PMA #P200040) platform originally approved in 2021, using enhanced imaging workflows and AI-powered analysis tools to improve breast tissue detection and characterization.
